A fun, interactive trivia game built with React. It fetches real-time questions from the OpenTDB API.

## Features
* **Dynamic Questions:** Fetches 5 random trivia questions per round.
* **Instant Feedback:** Highlights correct and incorrect answers in real-time.
* **Score Tracking:** Displays your final score at the end of the quiz.
* **Responsive Design:** Styled with CSS and Karla font for a modern look.

## Technologies Used
* React (Vite)
* [html-entities](https://www.npmjs.com/package/html-entities) (for decoding API data)
* [clsx](https://www.npmjs.com/package/clsx) (for conditional CSS)

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
